# Key

- use `markdown` insteadof `reStructuredText`
- use `pipenv`
- add `pytest` , `pytest-watch` in dev dependencies


# Usage
```
pip install cookiecutter
cookiecutter https://github.com/mildronize/cookiecutter-pypackage-minimal.git
pipenv shell
```

## Getting Started

Start dev

```
pipenv shell
make dev
```

run test

```
make test
```
